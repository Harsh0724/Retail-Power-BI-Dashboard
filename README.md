**Retail Sales Performance Analysis & Power BI Dashboard**
**Project Overview**
This project focuses on analyzing retail transaction data and building an interactive dashboard to visualize sales performance.
The dashboard was developed using Microsoft Power BI to transform raw data into meaningful insights that help understand product performance, sales trends, and regional performance.

The objective of this project is to demonstrate business intelligence and data visualization skills by converting transactional data into actionable insights.

**Dataset**

The dataset contains retail transaction data including:

Product information

Sales and revenue data

Customer transactions

Regional sales details

Product categories and performance metrics

The dataset was used to analyze sales patterns and identify high-performing products and regions.

**Tools & Technologies**

Microsoft Power BI

Power Query (Data cleaning and transformation)

DAX (Data Analysis Expressions)

Data Modeling

**Dashboard Features**

Interactive dashboard with filters and slicers

KPI indicators for sales and revenue

Visualizations showing sales trends over time

Product category performance analysis

Regional sales comparison

**Key Insights**

Identified the top-performing product categories contributing the highest revenue.

Analyzed monthly sales trends to identify peak sales periods.

Compared regional sales performance to highlight strong markets.

Highlighted top-selling products based on total sales.
